LA 2 总结

Jordan Normalform

计算Jordansche Normalform

  • CPΦ(X) := det(X*IdV - Φ)
  • Eig(Φ,λ) := Kern(Φ - λ*IdV)
  • 对于某个Eigenwert λ，关于λ 的Jordanblocks 的长度 = dim(Hauptraum zu λ), 也就是λ 的algebraischen Vielfachheit
  • 对于某个Eigenwert λ，关于λ 的Jordankästchen 是dim(Eig(Φ,λ)),也就是geometrische Vielfachheit
  • 对于某个Eigenwert λ，长度大于k 的Kästchen = dim Kk(λ) - dimK(k-1)(λ)

计算Jordanbasis (S: J = S^(-1)AS)

设：我们有两个eigenwert λ和ɲ，其中ɲ 的 algebraischen Vielfacheit = geometrische Vielfachheit. λ的不相等。

  1. Eig(Φ,ɲ) 的向量直接可以放进S
  2. 算出Hauptraum(Φ,λ) 的向量，选随机的的一个 H(λ)/Eig(λ)向量作为b1。
  3. 算出 (A-λIdV)*b1 = b2
  4. 重复步骤3， H 和Eig 的向量相差次
  5. 最后从Eig(λ) 补充不够的向量加入S，保证linear unabhängig
  6. 位置排序依照JNF
